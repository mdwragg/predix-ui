Welcome to Predix UI

We've produced Predix UI as a set of user interface components that enable designers and developers to quickly and easily create Industrial Internet web applications that run on top of Predix services and data.

Tried-and-true technology

Predix UI is implemented using modern web application technologies such as a powerful, scalable and Sass-based CSS framework. You'll find components powered by D3, the best of open source web tech and we're going all-in on web components. Predix UI intends to be framework-agnostic, so you can use Angular, Ember, Backbone, or pure Polymer.

Predix UI GitHub

Predix UI uses Google's Polymer, which provides innovative "syntactic sugar" and polyfill libraries for practical adoption of modern web components. Polymer helps build web applications with custom, encapsuled, reusable components that work just like HTML elements.
